# WebAuthn demo

Slides: https://slides.com/fidoalliance/jan-2018-fido-seminar-webauthn-tutorial#/

### Requirements

- Firefox Nightly
- Nodejs + NPM
- Text editor

### Run

- `git clone https://github.com/Hemalatha1011/Passwordless-Authentication-.git`
- `cd webauthn-demo`
- `npm install`
- `node app`
